ui/스타일은 기존 컴포넌트를 위주로 사용해서 제작하면돼.


디자인은 실제 사이트들의 로그인/회원가입 페이지 디자인을 이용해서 작업해줘.

id는 email형태로 입력받고 입력시 자동 validation 하는 기능 추가해줘.
아이디, 패스워드 입력받고 로그인 버튼이 있어야되고 아이디/비밀번호 찾기 버튼 / 회원가입 버튼이 있어야되고
아이디/비밀번호 찾기는 todo 개발로 미루고 현재는 로그인버튼/ 회원가입 버트만 만들어줘.
회원가입버튼 클릭시 회원가입 페이지로 이동한다음 아이디. 비밀번호 입력하게하고 아이디 유효성 검사는 email형태인지확인,. 비밀번호 유효성 검사는 영어,숫자, 특수문자가 포함되는지 확인정도 하면 될거같아. 회원가입 성공시 로그인 페이지로 자동 이동하게해주고 로그인 성공시 메인화면으로 이동하게 해줘.


이런식으로 생각하고있어. 네비게이션바나 부터 등은 기존의 페이지와 동일하게 맞춰서 작업하면 될거같아.
